About Food Rescue

Food Rescue System

Food Rescue System is a web-based platform designed to reduce food wastage by connecting extra food donors with people and animals in need. It provides a simple and organized way to donate usable food instead of letting it go to waste.

Project Goal

The main goal of this project is to reduce food waste and create a clean digital system where donors can register, log in, and donate food in an organized manner.

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python, Django
  • Database: MySQL
